Dotfiles using dotbot
=================

This is a dotfiles config with [Dotbot][dotbot].

Usage
-----------

```
git clone git@github.com:thiagolopes/dotfiles.git ~/.dotfiles
cd .dotfiles
./install
```

apply diff to model:
`git apply <model>.diff`
